# 📘 Master Project Knowledge Base (MPKB)

## CNC Digital Twin
### Real-Time Industrial IoT Monitoring & Hybrid AI Predictive Maintenance

---

## Purpose

This Master Project Knowledge Base (MPKB) serves as the authoritative technical reference for the CNC Digital Twin project.

Unlike the GitHub README, which provides a high-level overview, this knowledge base documents every significant aspect of the system, including architecture, implementation details, design decisions, execution flow, engineering trade-offs, production considerations, and interview preparation material.

The objective is to ensure **zero information loss**, allowing the project to be understood, maintained, extended, and confidently discussed in technical interviews without repeatedly inspecting the source code.

---

# Objectives

This knowledge base is intended to:

- Document the complete system architecture.
- Explain every important source file and module.
- Capture the reasoning behind engineering decisions.
- Describe implementation details that are not obvious from the code.
- Preserve the project's evolution from hardware prototype to software simulation.
- Serve as the single source of truth for resumes, interviews, portfolio, GitHub documentation, and future development.

# Guiding Principles

Throughout this knowledge base, every topic is documented from three perspectives:

## 1. Current Implementation

Describes exactly how the project works today.

No assumptions.

No exaggeration.

Only actual implementation.

---

## 2. Engineering Rationale

Explains why a particular design or technology was chosen.

Includes trade-offs, alternatives, and lessons learned.

---

## 3. Production Perspective

Describes how an industrial production system would implement or improve the same functionality.

This separation helps distinguish between:

- Portfolio implementation
- Production engineering
- Future improvements
